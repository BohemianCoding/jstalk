INSTALLING THE ACORN PLUGIN
--------------------------

The Acorn plugin requires Acorn: http://flyingmeat.com/acorn/
And you need to put the "JSTalk.acplugin" plugin inside this folder:
~/Library/Application Support/Acorn/Plug-Ins/



INSTALLING THE VOODOOPAD PLUGIN
------------------------------

The VoodooPad plugin requires VoodooPad: http://flyingmeat.com/voodoopad/
And you need to put the "JSTalk.vpplugin" plugin inside this folder:
~/Library/Application Support/VoodooPad/PlugIns/
